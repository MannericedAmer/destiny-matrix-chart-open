# Destiny Matrix Chart Open

Destiny Matrix Chart Open is a powerful tool designed to calculate and visualize your Destiny Matrix. By integrating numerology and astrology, this project provides a user-friendly interface to explore your personal energy and life path.

## Technologies Used

- CSS
- HTML
- JavaScript

## Features

- **Personal Calculation**: Uses birthdate and numerology to create a personalized Destiny Matrix chart.
- **Compatibility Matrix**: Allows users to input two birthdates to determine their energetic compatibility.
- **Responsive Design**: Optimized for both desktop and mobile viewing.

## How to Use

1. Clone this repository to your local machine.
2. Open `index.html` in any modern web browser to start a personal calculation.
3. Open `compatibility.html` to calculate compatibility between two people.
4. Enter the required birthdate(s) and click "Create Your Chart".
